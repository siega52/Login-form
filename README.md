# Login Form

Современная и адаптивная форма входа с социальными авторизациями.

## Демо

[Посмотреть демонстрацию](https://siega52.github.io/Login-form/)

## Описание

Чистая и современная форма аутентификации с поддержкой:
- Входа через email и пароль
- Социальной авторизации (Google, Apple)
- Адаптивного дизайна
- Доступности (ARIA attributes)

## Особенности

- **Современный дизайн** - чистый и минималистичный интерфейс
- **Адаптивность** - корректно отображается на всех устройствах
- **Социальные кнопки** - быстрый вход через Google и Apple
- **Валидация форм** - встроенная HTML5 валидация
- **Доступность** - поддержка screen readers и клавиатурной навигации

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - кастомные свойства (CSS Variables), Flexbox
- **Google Fonts** - шрифт Mulish
- **Адаптивный дизайн** - mobile-first подход
